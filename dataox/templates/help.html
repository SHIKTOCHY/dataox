What is RDF?

RDF is a data model in which facts are represented as a set of triples, each comprising a subject, predicate (verb), and object. Each element is either a URI (which, being uniquely identifying, allows everyone to talk about the same concept), an anonymous node, or a literal value. For example, "Alice knows Bob" might be expressed as: <http://example.org/alice> foaf:knows <http://example.org/bob>.

Predicates are always URIs and are usually drawn from commonly-understood vocabularies. By reusing vocabularies, information represented as RDF can be understood without <http://example.org/alice>
